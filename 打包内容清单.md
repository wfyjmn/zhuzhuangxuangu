## 📋 安装后需要做的事情

### 1. 创建 .env 文件（必须）

```bash
cd assets
cp .env.example .env
```

编辑 `.env` 文件，填入您的 Token：
```
TUSHARE_TOKEN=your_actual_token_here
```

### 2. 安装依赖（必须）

```bash
pip install -r requirements.txt
```

### 3. 运行测试（推荐）

```bash
cd assets
python -c "from config import TUSHARE_TOKEN; print('Token 配置成功')"
```

### 4. 运行程序

```bash
python main_controller.py full
```
