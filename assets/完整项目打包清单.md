使用说明
========================================

1. 解压缩
   tar -xzf DeepQuant_完整项目_20260131_220741.tar.gz

2. 创建 .env 文件
   echo "TUSHARE_TOKEN=your_token_here" > .env

3. 安装依赖
   pip install -r requirements.txt

4. 运行训练
   # V4.0.0 真实资金流训练
   python3 scripts/train_auto_optuna.py

   # 柱形突击训练
   python3 scripts/train_high_return_assault.py
