## 🚀 安装步骤

### 步骤 1: 下载压缩包

下载 `DeepQuant-System-v3.0-Final.tar.gz` 到本地

### 步骤 2: 解压

**Windows**:
- 使用 WinRAR、7-Zip 等解压软件
- 右键点击 → "解压到当前文件夹"

**Linux / macOS**:
```bash
tar -xzf DeepQuant-System-v3.0-Final.tar.gz
cd temp_package_final
```

### 步骤 3: 配置 Token（重要！）

压缩包中**不包含** `.env` 文件，需要手动创建：

```bash
cd assets
cp .env.example .env

# 编辑 .env 文件
# Windows: notepad .env
# Linux/macOS: nano .env
```

在 `.env` 文件中填入：
```
TUSHARE_TOKEN=your_actual_token_here
```

### 步骤 4: 安装依赖

```bash
pip install -r requirements.txt
```

### 步骤 5: 运行程序

```bash
cd assets
python main_controller.py full
```

## 🔧 故障排除

### 问题 1: 找不到 Token

**错误**: `ValueError: ❌ 未配置 Tushare Token！`

**解决**:
1. 确认 `assets/.env` 文件存在
2. 确认文件中包含 `TUSHARE_TOKEN=...`
3. 确认 Token 值正确

### 问题 2: 依赖缺失

**错误**: `ModuleNotFoundError`

**解决**:
```bash
pip install -r requirements.txt
```

### 问题 3: 程序无法运行

**可能原因**:
- Python 版本不兼容
- Token 无效
- 网络连接问题

**解决步骤**:
1. 检查 Python 版本
2. 验证 Token
3. 测试网络连接
