📦 包含的核心内容：

【源代码】(src/)
- stock_system/     : 核心系统模块（预测、分析、监控等）
- utils/            : 工具类（日志、文件、消息、辅助函数）
- storage/          : 存储（内存、数据库、对象存储）
- graphs/           : 图相关模块
- main.py           : 主入口

【脚本】(scripts/)
- train_with_real_data.py   : 使用真实Tushare数据训练
- train_with_mock_data.py   : 使用模拟数据训练
- generate_mock_data.py     : 生成模拟数据
- setup.sh                  : 环境设置脚本
- local_run.sh / http_run.sh: 运行脚本
- load_env.py / load_env.sh : 环境变量加载

【资源】(assets/)
- data/           : 训练数据和测试数据

- reports/        : 训练报告和可视化图表（HTML/PNG）
- cache/          : 数据缓存（行情、预测、日志）
- logs/           : 系统运行日志

【配置】(config/)
- model_config.json       : XGBoost模型配置
